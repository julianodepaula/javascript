Aula B8
* Operadores parte 2

    Relacionais:
        maior >
        menor <
        maior igual >=
        menor igual <=
        igual ==
        diferente !=

            Retornam True ou False
            5 > 2 é True

        Identidade: ===
            5 == 5 True
            5 == '5' True
            5 === '5' False
            5 === 5 True


    Lógicos:
        negação ! 'não'
        conjunção && 'e'
        disjunção || 'ou'

            Retornam True ou False
            true  && true  é True
            true  && false é False
            false && true  é False
            false && false é False

            true  || true  é True
            true  || false é True
            false || true  é True
            false || false é False

* Dica Ordem de Precedência:
    > Aritméticos + - * / % **
        > Relacionais < > >= <= == !=
            > Lógicos ! > && > ||

    Ternário:
        ? :
        teste ? true : false
        Ex: média >= 7.0 ? "Aprovado" : "Reprovado"
